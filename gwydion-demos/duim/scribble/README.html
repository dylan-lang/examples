Scribble Example Project

This is a simple DUIM example program for freehand line drawing. It demonstrates the following techniques:

  • Defining your own window class.
  • Defining your own handle-repaint method.
  • Handling mouse events.
  • Using some of the DUIM graphics functions.
  • Defining a DUIM frame.
  • Using a popup menu.

In Functional Developer Professional and Enterprise editions there is a companion example called OLE Scribble which demonstrates converting this DUIM application to an OLE control.
