Koala HTTP Server and Template Engine

Koala is an HTTP server written in Dylan. It includes Dylan Server Pages (DSP), a template engine similar to JSP, without scriptlets. But because it is written in Dylan, and can take advantage of Dylan's powerful language features, creating web apps in it should be much more concise and straightforward than with JSP.

The HTTP server portion of Koala was written mostly by Gail Zacharias. She moved on to other things before it was in a functioning state, so I finished it up and added Dylan Server Pages and XML-RPC.

The current version is v0.3. It is provided mostly for the interest and amusement of other Dylan programmers and is nowhere near production quality. The current version is for Windows only. I plan to fix this as soon as I can by porting to Gwydion Dylan. If you want to help, let me know!

There is a small demo application included, which shows how to use the features of Koala and DSP.
